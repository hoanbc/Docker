### Tạo image để triển khai cho máy khác
docker commit -m "Comment" -a "Tác giả"  [Container ID] [Image Name]
docker commit -m "Centos Project01" -a "HoanBC" d452f1a1b69d hoanbc/project01:v1

### Cho phép truy cập và điều khiển Docker Node từ xa
vim /lib/systemd/system/docker.service
From: ExecStart=/usr/bin/dockerd -H fd:// --containerd=/run/containerd/containerd.sock
Change To: ExecStart=/usr/bin/dockerd -H fd:// -H=tcp://0.0.0.0:2375 --containerd=/run/containerd/containerd.sock

### Đăng nhập vào docker/hub
docker login -u username -p password
docker login -u hoanbc -p devops123

### Đưa image lên docker hub để mọi người cùng sử dụng
docker push [Tên image]
docker push hoanbc/project01:v1

### Kiểm tra phiên bản và các thông số của Docker
docker -v
docker system info
docker-compose version
